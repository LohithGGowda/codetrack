# CodeTrack Project

This project, originally developed as part of a **DevOps Micro Internship (#DMIByPravinMishra)**, has been enhanced to serve as a personal coding journey tracker named "CodeTrack".

## Project Overview

CodeTrack is designed to help developers visualize their coding progress, set and achieve goals, and celebrate milestones in their learning journey. It provides a simple, clean interface to track various aspects of coding development.

## Features

- **Progress Visualization:** See your coding activity and growth over time with clear layouts.
- **Goal Setting:** Define and monitor personal coding goals.
- **Milestone Recognition:** Acknowledge and celebrate key achievements.
- **Contact Page:** A dedicated page for inquiries and feedback.

## Setup and Usage

This is a static website. To view it:

1.  Clone this repository to your local machine.
2.  Open the `index.html` file in your web browser.
